# Enable third party updates 

*Applies to: System Center Configuration Manager (Current Branch)*

Beginning with version 1806, the Third-Party Software Update Catalogs node in the Configuration Manager console allows you to subscribe to third-party catalogs, publish their updates to your software update point, and then deploy them to clients.
