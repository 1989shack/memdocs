## <a name="bkmk_extend"></a> Improvements to extend and migrate on-premises site to Microsoft Azure
<!--6307931-->
[Extend and migrate on-premises site to Microsoft Azure](/configmgr/core/support/azure-migration-tool) now supports provisioning multiple site system roles on a single Azure virtual machine. You can add site system roles after the initial Azure virtual machine deployment has completed.
